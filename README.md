# Посмотри в окно

Второй проект из курса «Практикум». Изначальный HTML-код был предоставлен готовым — задача заключалась только в написании CSS. Ключевой навык, который тренировался в проекте, — работа с частично готовым модулем (адаптация вёрстки под заданную структуру разметки, а не создание её с нуля).
